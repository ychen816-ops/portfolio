# Portfolio Development Session — 2026-04-07

## 1. Windowscape Shuffle Interaction (standalone)
**File:** `windowscape-shuffle.html`

Created an interactive page that loads `hero-image-3.png`, extracts all black graphic elements via connected-component labeling on canvas, and displays them as individual pieces. Each click randomly rearranges all elements with smooth CSS transitions.

- Used union-find algorithm for pixel-level element extraction
- 4 layout modes: scatter, circle, grid, wave
- Overlap avoidance algorithm (30 attempts per element)

## 2. Element Size & Speed Adjustments

## Design Decisions
- Palette consistency: yellow/blue/green bias for Windowscape, project-specific colors for Bias Bingo
- Anti-aliasing: 2x canvas rendering + smooth alpha falloff for extracted elements
- Staggered animations to avoid mechanical feel
- Per-character interaction over whole-element for more playful UX
- Goose-yellow (鹅黄) specifically requested for Balcony dots
- Acrylic token transparency preserved from source PNG
